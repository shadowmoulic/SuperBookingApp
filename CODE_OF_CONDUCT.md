# Code of Conduct

## 1. Purpose
This Code of Conduct outlines the standards of professional behavior expected from all team members, contractors, and partners working on SuperBookingApp. As a closed-source, proprietary project, our workspace, tools, and communications (including source code repositories, communication channels, and design documents) must remain professional, respectful, and collaborative.

## 2. Expected Behavior
- **Professionalism:** Treat all colleagues, partners, and stakeholders with respect and dignity.
- **Inclusivity:** Value diverse perspectives and foster an environment where everyone feels safe to contribute and share ideas.
- **Constructive Collaboration:** Focus code reviews, issue discussions, and architectural debates on technical merits and project goals rather than personal attributes.
- **Confidentiality:** Adhere to all non-disclosure agreements (NDAs) and information security policies. Project details, code, and customer data must not be shared outside of authorized channels.

## 3. Unacceptable Behavior
- Harassment, discrimination, or intimidation of any form.
- Disrespectful or derogatory language in code comments, pull request reviews, commit messages, or communication channels.
- Unauthorized sharing or leaking of intellectual property, code, or internal documentation.

## 4. Reporting Guidelines
If you witness or experience behavior that violates this Code of Conduct, please report it immediately to:
- Engineering Leadership or your direct manager.
- The Human Resources / People Operations department.

All reports will be treated confidentially and investigated promptly.
